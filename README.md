## ğŸŒ€ Parallax Effect Explained

The hero section uses a **3D parallax technique** by:

- Applying `perspective` to the parent container
- Placing multiple background layers at different **z-depths**
- Scaling layers based on their `translateZ()` values
- Creating a natural depth illusion while scrolling

This approach gives a smooth and immersive visual experience **without JavaScript**, relying purely on **CSS transforms and stacking**.
